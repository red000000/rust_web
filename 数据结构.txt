登录信息
路由:http://127.0.0.1:8080/sign_up
请求方式:POST
前端发送信息:
SignUpInfo:{
    username:string
    password:string
}

返回值:
    SignUpOk:{
        message:"登录成功!"
    }
    SignUpFail:{
        message:"登录失败!"
    }
注册信息
路由:http://127.0.0.1:8080/sign_in
请求方式:POST
前端发送信息:
SignInInfo:{
    username:string
    password:string
    sex:string
    birthday:string
    country:string
    province:string
    city:string
}
返回值:
    SignInOk:{
        message:"登录成功!"
    }
    SignInFail:{
        message:"登录失败!"
    }
