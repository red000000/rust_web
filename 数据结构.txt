页面信息
登录页面路由:http://127.0.0.1:8080/sign_up
注册页面路由:http://127.0.0.1:8081/sign_in
主页页面路由:http://127.0.0.1:8081/index

登录信息
路由:http://127.0.0.1:8081/dynamic/api/sign_up
请求方式:POST
前端发送信息:
SignUpInfo:{
    username:string
    password:string
}
后端返回值:
    成功:
    SignUpMessage:{
        message:string,
        message_type:u32,(0)(大于0,具体情况在sign_up_statement.rs中定义)
        flag:true,
    }
    失败:
    SignUpMessage:{
        message:string,
        message_type:u32,
        flag:false
    }

注册信息
路由:http://127.0.0.1:8081/dynamic/api/sign_in
请求方式:POST
前端发送信息:
SignInInfo:{
    username:string,
    password:string,
    sex:string,
    birthday:string,
    country:string,
    province:string,
    city:string,
}
后端返回值:
    成功:
    SignInInfoMessage:{
        message:string,
        message_type:u32,(0)
        flag:true,
    }
    失败:
    SignInInfoMessage:{
        message:string,
        message_type:u32,(大于0,具体情况在sign_in_statement.rs中定义)
        flag:false,
    }


请求用户信息api
路由:http://127.0.0.1:8081/dynamic/api/get_user_info
请求方式:GET
前端发送信息:
    SearchUserInfo{
        username:string,
    }
后端发送信息:
    成功找到用户信息:
    UserInfo:{
        username:string,
        password:string,
        sex:string,
        birthday:string,
        country:string,
        province:string,
        city:string,
        profile_photo:string,
    }
    失败:
    SearchUserInfoMessage:{
        message:"用户不存在!",
        flag:false,
    }

上传用户头像
路由:http://127.0.0.1:8080/static/api/upload_user_profile_photo
请求方式:POST
前端发送信息:

一个multipart/form-data编码的表单,内部包括两个Blob块:

    UploadUserProfilePhoto{
        username:string,
        profile_photo_url:string,
    }
    type:application/json

    图像文件二进制数据
    type:multipart/form-data

后端发送信息:
    成功:
    UploadUserProfilePhotoMessage:{
        message:string,
        message_type:u32,(0)
        flag:true,
    }
    失败:
    UploadUserProfilePhotoMessage:{
        message:string,
        message_type:u32,(大于0,具体情况在upload_user_profile_photo_statement.rs中定义)
        flag:false,
    }

//静态服务器使用路由,与前端无关
