首页{
   // 展示不同分类的分区信息，加入分区可以浏览帖子，选择性保存（关注）分区，保存（关注）的分区会将分区头像（超链接）添加到左侧导航栏
    用户未登录默认开启部分： 分区信息展示，无法进入分区
    分区块{
        分区头图： 展示分区图像内容，图片格式，
        分区头像： 作为分区头像展示，图片格式，
        分区名： 分区的名称，文本格式，
        分区简介： 展示分区内容，文本格式，
        分区热度： 展示分区受关注度，文本格式。
        }
    用户头像： 用户自设，未登入为默认图片，左侧常驻
    搜索： 搜索分区内容（查询总名称，返回分区块） 能做就做吧，不能就算了

分区{
    //展示具体分区，主页的发现分区导航栏替换为分区导航栏，推荐分区替换为展示分区帖子
    //用户未登录默认不开启
    分区头像： 左侧导航栏常驻 图片格式
    分区头图： 分区导航栏常驻 图片格式
    分区总名： 分区名，分区导航栏常驻，文本格式
    分区分名： 分区导航栏的不同讨论区的名称，文本格式
    帖子块{
        帖子名： 用户发布在分区内的内容的总标题
        时间： 记录帖子发布时间，用以对比时刻差
        帖子内容： 用户编辑的内容，文本格式、图片格式 图片可查看大图，这个好像是前端的活
        点赞； 记录帖子点赞数
        评论： 记录评论数 点击帖子背景和评论可进入帖子详情 这个也是前端的活
        帖子用户头像： 帖主及评论用户的头像，图片格式 点击头像展示用户简易信息（用户头像，用户名和性别 后端）和私信功能
        回复帖子： 用户回复的帖子，文本格式、图片格式
        }
}
